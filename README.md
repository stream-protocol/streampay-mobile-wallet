## Stream Mobile Wallet

A Stream Mobile Wallet in flutter

Features:
- Add New Wallet
- Import wallets with it's seedphrase (devnet only for now)
- Create new wallets (devnet only for now)


### Building android

```
flutter build android
```

### Building iOS

```
flutter build ios
```
